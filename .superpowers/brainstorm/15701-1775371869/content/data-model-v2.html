Bank Statement Reconciliation Logic (PMS)

Match Rule
Bank Debit → expect SecurityBought
Bank Credit → expect SecuritySold or DividendReceived

Match on: date ± 2 days + amount ± 0.5%
On Mismatch
→ Write ReconciliationFlagged event
→ Notify user: "₹X debit on DD/MM has no matching transaction"
→ User resolves: link or mark as non-investment
