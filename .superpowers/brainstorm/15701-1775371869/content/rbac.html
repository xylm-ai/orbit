Entity-Scoped Access

Advisors and CAs are granted access per entity, not the whole family. An advisor managing one entity cannot see another entity's portfolio unless explicitly granted.

family_user_access
user_id · entity_id · role · granted_by · granted_at
Owner always has access to all entities.
Invited users see only explicitly granted entities.

Inbound Email Routing

Each family gets a dedicated inbound address:
sharma-family@in.orbitwealth.in
Any authorised user (owner / advisor / CA) can forward statements to this address. Postmark parses the email, extracts attachments, and routes to the correct ingestion job. Sender email validated against users table — unknown senders rejected.
